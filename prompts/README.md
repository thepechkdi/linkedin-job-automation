# Prompts Folder

This folder contains all prompts used in the Job Search Automation project.

Files:
- **structure_resume.md** → Used to convert CV text to structured JSON
- **skill_match.md** → Used for job–resume matching + cover letter generation
- **cover_letter.md** → Standalone cover letter generator
- **extract_job_data.md** → Extract structured data from job postings
- **caching_logic.md** → Notes about duplicate prevention logic

These prompts are used inside n8n OpenAI nodes and can be updated anytime to improve accuracy.
